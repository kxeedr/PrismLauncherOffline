# Prism Launcher Offline
> ⚠️ Unofficial fork — not affiliated with the original Prism Launcher project.

## 📌 About This Fork

This project is a fork of the original **[Prism Launcher](https://github.com/PrismLauncher/PrismLauncher)**, modified to allow the use of **offline accounts** without requiring a valid Microsoft/Mojang account.

The primary goal of this fork is to remove the restriction that prevents launching the game without an authenticated online account.
